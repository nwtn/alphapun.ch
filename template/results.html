Below are: your original image; the mask alphaPun.ch generated; example HTML, CSS, and JS that you can use; and instructions for how to put these all together.

Copy below and paste into your site, or download a .zip file that contains everything you need.

  1. Reference the image in your HTML. You can use the generated example markup, or alter it to suit your needs.
    • The img and the links behind it must be in a container element with the class alphapunch.
    • If you run into trouble positioning your img and links, try putting the img into its own container element, like figure or div.
    • All links must have the class alphapunch-target, and must have an ID. Make note of these IDs, as youâ€™ll need them to configure your JavaScript.
    • If you run into trouble positioning your links, try putting them in a container element, like nav, ul, or div.
  2. Style your image and links. You can use the generated example styles, or alter them to suit your needs.
    • For alphaPun.ch to work, the stacking order from back-to-front must be:
      1. links (e.g. .alphapunch-target)
      2. image (e.g. .alphapunch img)
      3. link masks (e.g. .alphapunch-mask span)
      4. image masks (e.g. img + .alphapunch-mask span)
    • Internet Explorer requires the mask elements (e.g. .alphapunch-mask span) to have a background in order to properly register click events.
  3. Call the generated JavaScript. This should not be modified.
